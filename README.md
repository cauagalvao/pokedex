Pokédex

Uma Pokédex interativa desenvolvida com HTML, CSS e JavaScript, consumindo dados da PokéAPI.

Funcionalidades:
Buscar Pokémon por nome ou número
Navegar entre Pokémon
Exibição de sprite animado (quando disponível)

Tecnologias utilizadas:
HTML
CSS
JavaScript
PokéAPI

Voce pode encotrar o site em: pokedex-142.vercel.app
